HTML Boilerplate Definitions

Doctype

The Document Type Declaration, or doctype, tells the browser what type of document it is looking at. It is always the first item at the top of any HTML file.

The html element

The html element defines the whole document with a start and end tag. The lang attribute specifies the language and no-js/js specifies if JavaScript is enabled/disabled.

The head element

The head element is a container for metadata. The first line defines the character encoding for the document (utf-8). Another key part of the head is the stylesheet, which is included using the link element.

Title

The title element specifies the page title and is the only mandatory element inside the head.
